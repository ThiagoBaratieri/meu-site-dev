O Que é Entropia?

Na segurança da informação, entropia é a medida de aleatoriedade de uma senha. Quanto maior a entropia, mais difícil é para um algoritmo adivinhar a combinação.

Uma senha como senha123 tem baixíssima entropia porque usa palavras de dicionário e sequências lógicas. Já Xj9#mP2! é curta, mas tem alta entropia.

A Regra dos 4 Tipos

Para criar uma senha realmente forte, você deve misturar quatro tipos de caracteres:

  • Letras Maiúsculas (A-Z)
  • Letras Minúsculas (a-z)
  • Números (0-9)
  • Símbolos Especiais (!@#$%)

Adicionar apenas um símbolo especial pode aumentar o tempo de quebra de senha de algumas horas para séculos.
